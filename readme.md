## 关于本项目

laravel5.4+vue2.0开发的顺风车及后台,前后端分离,RESTful架构,RESTful只要维护资源的状态，而不需要维护客户端的状态。新手练习使用。

## 项目依赖

1 后端
- Laravel 5.4
- 后台基于Laravel-Admin
- RESTful API：[dingo/api](http://laravelacademy.org/post/3822.html)
- JWT认证

2 前端
- VUE2.0
- webpack
- vue-route
- vue-resource
- [VUX组件](https://vux.li/#/)基于WeUI和Vue(2.x)开发的移动端UI组件库，主要服务于微信页面。
